# RubyHtmlToPdf

A Ruby gem to convert HTML into PDFs using [Puppeteer](https://github.com/puppeteer/puppeteer).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruby_html_to_pdf'
```
